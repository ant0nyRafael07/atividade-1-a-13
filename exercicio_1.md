# Exercício 1 - Proposições Lógicas

Informe o valor lógico (verdadeiro ou falso) para cada proposição:

a.  ( **Verdadeiro** ) 2 \* 4 == 24/3\
    Explicação: 2 \* 4 = 8 e 24/3 = 8, então são iguais.

b.  ( **Falso** ) (15 % 4) \< (19 % 6)\
    Explicação: 15 % 4 = 3 e 19 % 6 = 1. Como 3 \< 1 é falso.

c.  ( **Verdadeiro** ) nao(2 \< 5) != (8 == 8)\
    Explicação: nao(verdadeiro) = falso e 8 == 8 é verdadeiro. Falso é
    diferente de verdadeiro.

d.  ( **Verdadeiro** ) verdadeiro ou falso\
    Explicação: No operador "ou", basta uma condição ser verdadeira.

e.  ( **Verdadeiro** ) 2*2 == Matematica.potencia(2, 2)\
    Explicação: 2*2 = 4 e 2² = 4.

f.  ( **Falso** ) verdadeiro e ((7 / 2) \> 3.5)\
    Explicação: 7/2 = 3.5. A comparação 3.5 \> 3.5 é falsa.

g.  ( **Verdadeiro** ) (Matematica.arredondar(2.8, 0) == 3) e
    (Matematica.raiz(9, 2) == 3)\
    Explicação: 2.8 arredondado para 0 casas fica 3 e raiz quadrada de 9
    é 3.
